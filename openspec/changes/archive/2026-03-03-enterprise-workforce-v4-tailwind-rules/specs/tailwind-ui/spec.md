## ADDED Requirements

### Requirement: Tailwind CSS Integration

The system MUST use Tailwind CSS for all frontend styling.

#### Scenario: Rendering the main layout

- **WHEN** the user loads the application
- **THEN** the layout is styled using Tailwind CSS utility classes rather than Bootstrap.

### Requirement: Modern UI Components

The system MUST replace legacy standard HTML tables with Tailwind-styled Cards for data presentation.

#### Scenario: Viewing data lists

- **WHEN** the user views a list of records
- **THEN** the records are displayed as a responsive grid of Cards with shadows and hover transitions.
